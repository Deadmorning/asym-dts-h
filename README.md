# Asym-DTS H 版策略

中证 500 ETF (510500) 基于 000905.SH 指数的双层趋势跟踪策略。

**回测表现**：2023-04-17 ~ 2026-04-15，10 万初始资金 → **+64.80%（97 笔交易）**，相对纯 WTS 基线 +13.33pp。

---

## 核心设计

两层信号的分工：

- **周线 WTS（战略层）**：决定是否允许做多
- **日线 DTS（战术层）**：在多头周内做短线进出决策

H 版状态机：

```
if wts == 0:                      空仓
elif 新多头周 (last_wts=0→1):      强制入场   ← V 底捕获
elif UP+UP 强多头周:               强制持仓   ← 主要 alpha 来源
elif DOWN+DOWN:                   空仓      ← 结构对称
else:                              DTS 双向   ← 震荡市套利
```

相对原 BigQuant v2 修复了 6 处结构问题（D_REV 量级、前一日取法、WTS 短周过滤、WTS 缓存、last_wts 周更、UP+UP 保护）。

详见 [REPORT.md](REPORT.md) 和 [COMPARISON.md](COMPARISON.md)。

## 策略定位

**一句话定位**：Asym-DTS H 是一个**基于散户情绪周期的、多时间框架状态机驱动的、带非对称风控的择时策略**。

- **目标市场**：A 股羊群效应显著的中盘 ETF（510500 最匹配 > 510300 > 50ETF）
- **目标场景**：单边趋势市跟上节奏、震荡市保本、接受 V 型极端反转吃不到
- **它不是**：
  - 不是纯左侧（抄底赌均值回归）→ 是"左侧抓反弹 + 右侧确认趋势"的混合
  - 不是趋势突破（海龟那种）→ 是"状态切换驱动 + 多时间框架协同"
  - 不是躺平 B&H → 牛市"表现为"持仓不动，但识别趋势是设计目的，不动只是结果

### 使用方法（一张图）

```
市场阶段          策略行为                H 版状态
────────────────────────────────────────────────────
牛市（确立）      全仓持有，接近 B&H      UP+UP 保护 / new_bull
熊市起始          DTS 双向少量交易        else / DTS 分支
震荡市            DTS 反转信号套利         else / DTS 分支
V 型反转          可能吃不到首周反弹      接受（见"失效场景"）
连续单边跌         空仓等待                DOWN+DOWN
```

核心价值：在熊市开始时用"少量交易摩擦"换"牛市到来时跟上节奏"的能力，解决纯左侧策略"不知道何时止损"和纯右侧策略"牛市错过启动段"的双重难题。

## Alpha 来源（不止羊群效应）

策略同时利用多个认知偏差，属于 A + C + D 混合型（不是 bug，是多时间框架互相校验）：

| 行为偏差 | H 里怎么用 |
|---|---|
| 羊群效应 (Herding) | DTS 3 连涨/跌的追随信号 |
| 过度反应 (Overreaction) | DTS 反转日 ← 急跌后反弹特别强 |
| 锚定效应 (Anchoring) | 7 规则里 prev_close vs curr_open = 周心理锚点 |
| 处置效应 (Disposition) | 套牢盘形成的特定反弹形态 |
| 损失厌恶 (Loss Aversion) | 恐慌下跌后的"解套情绪反弹"强度放大 |
| 多时间框架盲点 | 周 K 定状态 + 日 K 定时点，单一尺度看不全 |
| 反身性 (Reflexivity) | UP+UP 保护期 = 承认上涨自我强化 |

### Alpha 类型与持久性

| 类型 | 占比 | 持久性 | 说明 |
|---|---|---|---|
| **Behavioral Alpha** | 主要 | **High** | 神经认知限制不会被学会消除 |
| **Structural Alpha** | 次要 | Medium | T+1、涨跌停板塑造的反弹形态；制度变则变 |
| **Information Alpha** | 无 | — | 好事：不怕被人发现抄走 |

## "Asym"（非对称）设计含义

v2 名字里的 "Asym" 指两层非对称：

1. **时间非对称**：UP+UP 保护期宽容（最多 4 周），DOWN+DOWN 立即空仓 → 反映"上涨慢、下跌快"的市场特性
2. **阈值非对称**：D_REV / W_REV 可独立调参，反映 A 股反弹与回落的不同强度

这是 v2 区别于 v1 的核心哲学，也是为什么能在震荡下行中保本的结构性原因。

## 参数

当前校准值（2023-04 ~ 2026-04 三年期最优）：

| 参数 | 值 | 说明 |
|---|---|---|
| `W_FLAT` | 0.018 | 周 K FLAT 门槛（1.8%） |
| `W_SAME` | 0.009 | 周 K 同向差门槛 |
| `W_REV`  | 0.012 | 周 K 反向差门槛 |
| `D_FLAT` | 0.007 | 日 K FLAT 门槛（0.7%） |
| `D_SAME` | 0.003 | 日 K 同向差门槛 |
| `D_REV`  | 0.003 | **日 K 反向差门槛（DTS 灵敏度关键）** |

建议实盘每 6-12 个月用滚动窗口重新校准 `D_REV`，其他参数在合理区间内相对稳定。

## 为什么 H 版每个分支都必要

消融实验（见 COMPARISON.md 第四节）：

| 变体 | 收益 | 结论 |
|---|---|---|
| H 版（完整） | **64.80%** | baseline |
| 去掉 UP+UP 保护（= GitHub asym_v2） | 40.39% | 主要 alpha 来源 |
| 新多头周 DTS 确认入场（L 版） | 30.25% | 漏掉 V 底 |
| DTS 只出不进（非对称） | 43.35% | 震荡市套利被阉割 |
| UP+UP 放开 DTS 出场 | 34.38% | T+1 下反而磨损加剧 |
| 部分非对称 | 48.03% | 多输入少输出 |

## 失效场景（实盘前必须理解）

按 strategy-framework：**如果写不出失效条件，说明还没真懂机制**。

| 失效类型 | 场景 | 表现 | 会恢复吗 | 应对 |
|---|---|---|---|---|
| Cyclical | 长期横盘震荡 | DTS 来回挨打，摩擦累积 | ✅ 下一轮方向市恢复 | 忍住不动策略 |
| Cyclical | V 型极端反转（如 2024-09-27） | 跳过情绪发酵阶段，策略吃不到首周反弹 | ✅ 10 年一遇，接受 | 不为个案过拟合 |
| Cyclical | 连续多周 DOWN 无反弹 | WTS 不发 UP，空仓；V 反时跑输 | ✅ 下一轮恢复 | 坚持纪律 |
| Structural | A 股机构化加速 / T+0 / 做空便利化 | 散户情绪周期被平滑，Alpha 衰减 | ❌ 可能永久性 | 重评估机制，考虑退场 |

**核心纪律**：理解失效原因 ≠ 修复失效。Cyclical 失效期要**忍住**不动策略（这是择时策略的"天花板税"），Structural 失效期要**主动**重新评估机制假设。

## 当前设计的局限（已知待优化项）

**信号层强，执行层弱**。下列海龟交易法则要素 H 版尚未吸收：

- **ATR 动态仓位**：当前全仓/空仓二元，未按波动率缩放
- **硬止损**：依赖 WTS 状态切换退出，无 ATR 跟踪止损兜底
- **Pyramiding**：一次性建仓，未按 0.5×ATR 分批加仓

### 容量上限

基于 510500 日成交量（5-20 亿），估计策略可容纳规模 ≈ **几千万 ~ 2 亿**。超过会有明显市场冲击，Alpha 被自己吃掉。

---

## 风险提示

- 策略在 A 股特殊机制（T+1、涨跌停、节假日）下设计，**不直接迁移**到美股/加密/商品
- 不能躲"单日情绪极端反转"黑天鹅（典型例：2024-10-09 开盘进场 → 后续暴跌 -7,749）
- V 型反转吃不到首周大涨（典型例：2024-09-27 那周，中秋+国庆短周过滤导致情绪链路中断）
- 回测数据含 2026-04-19 之后部分区间，参数校准不要使用未来数据
